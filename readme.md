# Добро пожаловать в Taurus!

> **Внимание!**
> Данный проект является хобби и не предназначен для использования в продакшене.
> Это может измениться в будущем, но на данный момент этот проект не даёт никаких гарантий.

Taurus - это IdP (Identity Provider), написанный на C# и разработанный,
чтобы быть лёгким и производительным. 

Несмотря на то, что Taurus является хобби-проектом, я стремлюсь выполнить его
качественно, пригодным к реальному применению.

На данный момент проект находится в ранних стадиях разработки.

Особенности:
- Поддержка SQLite
- Поддержка PostgreSQL
- Конфигурация сервиса в одном .yml файле
- Поддержка перезаписи конфигурации префексированными ENV-переменными
